# Project Overview
This project was completed as part of the Digital Literacy course. As a Student Digital Ambassador, the aim was to create awareness about digital tools, safe online practices, and professional communication.
The project includes five tasks covering infographic creation, digital portfolio building, platform exploration, email etiquette, and cybercrime awareness. Each task helped in developing practical digital skills that are useful for both academics and future career growth.
